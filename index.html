HTML — Structure of the Web

HyperText Markup Language — the backbone that gives every webpage its structure and meaning.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create the structure of web pages. It uses elements and tags to define different parts of a web page such as headings, paragraphs, images, links, and forms.

Key Concepts:
  • Elements & Tags — Building blocks like <h1>, <p>, <div>
  • Attributes — Extra info like id, class, href, src
  • Semantic HTML — Meaningful tags: <header>, <nav>, <main>, <footer>
  • Forms — User input collection with <form>, <input>, <button>
  • Tables — Structured data with <table>, <tr>, <td>
  • Media — Images & video: <img>, <video>, <audio>

CSS — Styling the Web

Cascading Style Sheets — giving HTML its visual identity, layout, and animations.

What is CSS?

CSS stands for Cascading Style Sheets. It controls how HTML elements look on screen — from colours and fonts to layout and animations. Without CSS, websites would just be plain text on a white page.

Key Concepts:
  • Selectors — Target elements: .class, #id, element
  • Box Model — Margin, Border, Padding, Content
  • Flexbox — One-dimensional layout
  • CSS Grid — Two-dimensional layout
  • Pseudo-classes — :hover, :focus, :nth-child
  • Animations — @keyframes and transition
  • Media Queries — Responsive design
  • CSS Variables — Reusable values with --variable

Bootstrap — CSS Framework

The world's most popular frontend framework — build responsive, mobile-first websites faster.

Why I find Bootstrap valuable

Before learning Bootstrap, every responsive layout required writing extensive CSS media queries by hand — a slow and error-prone process. Bootstrap solves this by providing a consistent, mobile-first grid system and a library of pre-built components that work out of the box. Rather than reinventing common UI patterns like navbars, modals, and alerts, I can focus on the unique parts of my project. The utility classes (d-flex, gap-3, text-center, etc.) have also taught me to think about layout in a structured, systematic way. This portfolio itself is built on Bootstrap 5 — every section uses the grid, breakpoints, and component classes to achieve a responsive layout without a single media query written from scratch.

What is Bootstrap?

Bootstrap is a free, open-source CSS framework developed by Twitter. It provides a collection of pre-built CSS classes, components, and JavaScript plugins that make building responsive websites faster and easier.

Instead of writing CSS from scratch, Bootstrap provides ready-to-use classes like d-flex, text-center, or btn btn-primary.

Key Features:
  • Grid System — 12-column responsive layout
  • Utility Classes — Spacing, colours, display helpers
  • Components — Navbar, Cards, Modals, Alerts, Badges
  • Breakpoints — sm, md, lg, xl, xxl
  • JS Plugins — Dropdown, Carousel, Collapse, Toast
  • Mobile-First — Designed for small screens first

Bootstrap Accordion

Accordions use Bootstrap's collapse plugin to show/hide content panels — no custom JS needed.

Bootstrap divides the page into 12 columns. You use classes like col-md-6 to specify how wide an element should be at each breakpoint — enabling fully responsive layouts.

Bootstrap provides pre-built responsive utilities, consistent cross-browser behaviour, and dozens of ready-to-use components — saving hundreds of hours compared to building everything from scratch.

Mobile-first means you design for small screens first, then use media queries to progressively enhance the layout for larger screens. Bootstrap's breakpoints follow this approach — col-sm targets phones, col-lg targets desktops.

Bootstrap Toast Notifications

Bootstrap Modal Component

Modals are one of Bootstrap's most powerful components. They create overlay dialog boxes without writing any custom JavaScript — Bootstrap handles the show/hide behaviour entirely through data-bs-toggle attributes and its built-in JS bundle.

<!-- Trigger button - no JS needed! -->
<button data-bs-toggle="modal" data-bs-target="#myModal">Open Modal</button>

<!-- Modal markup -->
<div class="modal fade" id="myModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">...</div>
      <div class="modal-body">...</div>
      <div class="modal-footer">...</div>
    </div>
  </div>
</div>
Bootstrap Modal — Demo

Why use Bootstrap Modals?
  • Zero JS required — Purely driven by HTML data attributes
  • Accessible — ARIA attributes and keyboard focus are handled automatically
  • Responsive — Works perfectly on all screen sizes
  • Customisable — Sizes: modal-sm, modal-lg, modal-xl

JavaScript — The Web's Language

The programming language of the web — adding interactivity, logic, and dynamic behaviour.

What is JavaScript?

JavaScript (JS) is a lightweight, interpreted programming language that runs in the browser. It is used to make web pages interactive — from simple button clicks to complex single-page applications.

JavaScript is the only programming language that runs natively in all browsers without any installation.

Key Concepts:
  • Variables — let, const, var
  • Data Types — String, Number, Boolean, Array, Object
  • Functions — Regular & Arrow functions
  • DOM Manipulation — getElementById, querySelector
  • Events — onclick, addEventListener
  • Conditionals — if/else, switch
  • Loops — for, while, forEach
  • Fetch API — Async HTTP requests

JavaScript Fetch API — Getting Data from the Web

The Fetch API allows JavaScript to make asynchronous HTTP requests to retrieve data from external sources (APIs) without reloading the page.

// Fetch API Example
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error(error);
  });
